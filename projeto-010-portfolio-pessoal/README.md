# Portfolio Pessoal - Yuri Santos

Portfolio responsivo desenvolvido com HTML, CSS e JavaScript vanilla.

## 🛠️ Tecnologias

- HTML5 semântico
- CSS3 (Flexbox, Grid, responsivo)
- JavaScript ES6

## ⚡ Funcionalidades

- Navegação suave entre seções
- Dark mode toggle
- Formulário de contato
- Design responsivo
- 5 projetos showcase

## 🚀 Como executar

Abra o arquivo `index.html` no navegador
